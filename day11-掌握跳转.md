# 掌握跳转

## 标记
`m` + 小写字母 标记单文件 通常使用`mm`

`m` + 大写字母 标记多文件 通常使用`mM`

为了便于记忆通常只标记一个就可以了,标记多个会增加思考时间

## 跳转
`'`跳转到标记的行
`反引号` 跳转到标记的具体位置
通常使用'即可,因为更加好按,更好操作

# 跳转到定义

`gd` 即 jump to definition
很好理解了

# 跳转
任何一个大于一个单词或当前行导航的移动都算作一次跳转

vim会记录跳转
使用:jumps命令查看跳转历史 // 通常不用

向前跳: `ctrl + o`
向后跳: `ctrl + i`

最大使用场景: gd 进去看定义后跳回当前文件
