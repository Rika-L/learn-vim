# 掌握窗口的管理

新建窗口 `ctrl + w + v` 向右新建一个窗口
         `ctrl + w + s` 向下新建一个窗口

切换窗口 `ctrl + w + hjkl` 使用方向键切换

`ctrl + w + w` 切换到下一个窗口

`ctrl + w + c` 关闭窗口 这里只会关闭一个窗口

`ctrl + w + o` 只保留当前窗口 关闭其他窗口
